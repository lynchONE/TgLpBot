# 盈利曲线显示优化

## 更新日期: 2026-01-12

### 问题描述
Miniapp中Auto任务的盈利曲线图表上，每个开仓/平仓事件都显示完整的文字标注（如"USDT/WM · 开仓 $200.00"），
当交易次数多时，文字重叠导致图表非常拥挤，无法清晰查看曲线。

### 解决方案
修改 `miniapp/src/components/AutoPnLCurveCard.jsx` 中的marker配置：

1. **移除文字标注**：将marker的 `text` 属性设为空字符串
2. **保留箭头图标**：
   - 开仓：蓝色向上箭头 (#60a5fa)
   - 盈利平仓：绿色向下箭头 (#10b981)
   - 亏损平仓：红色向下箭头 (#ef4444)
3. **详细信息通过列表查看**：底部的"开/平仓记录"列表保留完整的交易对名称、时间和金额信息

### 修改的文件
- `miniapp/src/components/AutoPnLCurveCard.jsx`
  - 第58-90行的 `markers` useMemo，移除了构建文字标注的逻辑

### 效果
- 图表更加清晰，曲线可以完整显示
- 用户可以通过箭头颜色快速区分开仓/盈利/亏损
- 详细交易信息可在图表下方的记录列表中查看
