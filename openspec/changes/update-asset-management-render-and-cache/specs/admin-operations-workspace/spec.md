## ADDED Requirements

### Requirement: 资产管理管理员页签按需装载
系统 SHALL 在 `miniapp` 与 `webapp` 的资产管理模块中，对管理员页签采用按需装载策略，避免未激活的管理员工作区阻塞普通资产页面的首屏加载。

系统 MUST：
- 仅在管理员切换到对应页签后装载“运行管理”或“系统管理”的代码与数据
- 在用户停留于“我的资产”或“聪明钱资产”页签时，不得提前请求未激活管理员页签的数据
- 在管理员切回先前页签时，允许复用最近一次成功结果并后台静默刷新

#### Scenario: 普通用户打开资产管理模块
- **WHEN** 普通用户进入资产管理模块
- **THEN** 系统不得装载仅管理员可见的“运行管理”与“系统管理”数据请求

#### Scenario: 管理员尚未切换到运行管理页签
- **WHEN** 管理员停留在“我的资产”或“聪明钱资产”页签
- **THEN** 系统不得提前发起“运行管理”或“系统管理”的数据请求

#### Scenario: 管理员切换到系统管理页签
- **WHEN** 管理员切换到“系统管理”页签
- **THEN** 系统才装载该页签所需代码和数据
- **AND** 返回其它页签后不应阻塞当前资产视图的刷新
