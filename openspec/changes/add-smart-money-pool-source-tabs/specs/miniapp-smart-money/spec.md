## ADDED Requirements

### Requirement: 聪明钱池子视图必须支持钱包来源范围切换
聪明钱池子视图 MUST 允许用户在活跃池子列表中按聪明钱钱包来源切换展示范围。系统 MUST 至少提供 `全部`、`手动添加`、`合约发现` 三个范围，其中 `全部` 保持当前按所有聪明钱钱包聚合的行为。

#### Scenario: 默认展示全部聪明钱池子
- **WHEN** 用户进入聪明钱池子视图
- **THEN** 系统默认选择 `全部`
- **AND** 池子列表按所有活跃聪明钱钱包聚合展示

#### Scenario: 切换到手动添加范围
- **WHEN** 用户点击 `手动添加`
- **THEN** 系统只展示手动添加钱包参与的池子
- **AND** 钱包数、仓位数、聪明钱仓位金额和区间聚合 MUST 仅基于手动添加钱包计算

#### Scenario: 切换到合约发现范围
- **WHEN** 用户点击 `合约发现`
- **THEN** 系统只展示合约发现钱包参与的池子
- **AND** 钱包数、仓位数、聪明钱仓位金额和区间聚合 MUST 仅基于合约发现钱包计算

#### Scenario: 范围切换保留既有筛选能力
- **WHEN** 用户切换来源范围后继续使用搜索、协议筛选、金额筛选或费率筛选
- **THEN** 系统 MUST 在当前来源范围内应用这些筛选条件
- **AND** 分页 MUST 从第一页重新开始
