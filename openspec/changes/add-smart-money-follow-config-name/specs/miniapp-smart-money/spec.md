## ADDED Requirements

### Requirement: 自动跟单表单支持任务名称
MiniApp 和 WebApp SHALL 在自动跟单配置表单中提供任务名称输入，让用户为配置添加说明。配置列表和相关跟单任务展示 MUST 优先显示任务名称；未填写任务名称时 MUST 继续使用目标钱包或钱包组信息展示。

#### Scenario: 用户新增自动跟单配置时填写任务名称
- **WHEN** 用户打开自动跟单配置表单
- **THEN** 表单 MUST 提供任务名称输入
- **AND** 保存配置时 MUST 将任务名称提交给后端

#### Scenario: 用户编辑已有任务名称
- **GIVEN** 自动跟单配置已有任务名称
- **WHEN** 用户打开编辑表单
- **THEN** 表单 MUST 回填该任务名称
- **AND** 用户保存后配置卡片 MUST 展示更新后的名称

#### Scenario: 未填写任务名称
- **WHEN** 自动跟单配置没有任务名称
- **THEN** 配置卡片 MUST 继续展示目标钱包地址、钱包组数量和触发规则
- **AND** 页面 MUST NOT 显示“未命名任务”等会被误认为真实名称的占位值
