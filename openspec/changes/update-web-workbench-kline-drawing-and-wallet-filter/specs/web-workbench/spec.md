## ADDED Requirements

### Requirement: Web workbench K 线支持价格测量工具
Web workbench SHALL 在自绘 K 线面板中提供直线测量与矩形测量两种工具，用于读取价格区间的涨跌幅百分比。

#### Scenario: 用户使用直线测量
- **WHEN** 用户在 K 线工具栏中切换到直线工具，并依次选择两个点
- **THEN** 图表显示连接两点的测量线
- **AND** 图表显示基于两点价格计算出的涨跌幅百分比

#### Scenario: 用户使用矩形测量
- **WHEN** 用户在 K 线工具栏中切换到矩形工具，并拖拽出一个矩形区域
- **THEN** 图表显示该矩形区域
- **AND** 图表显示基于矩形上下边价格计算出的涨跌幅百分比

#### Scenario: 用户清除当前测量
- **WHEN** 用户切换回普通浏览模式、重新绘制，或点击清除测量
- **THEN** 当前测量结果被移除
- **AND** 图表恢复普通交互状态

### Requirement: Web workbench 支持钱包气泡筛选
Web workbench SHALL 在 K 线左上角提供钱包气泡筛选入口，允许用户按开单金额和钱包选择过滤当前图表上的聪明钱气泡。

#### Scenario: 用户按开单金额筛选气泡
- **WHEN** 用户设置金额阈值
- **THEN** 图表仅显示 `estimated_usd` 满足阈值条件的钱包气泡

#### Scenario: 用户按钱包勾选筛选气泡
- **WHEN** 用户在筛选面板中勾选一个或多个钱包
- **THEN** 图表仅显示属于这些钱包的气泡
- **AND** 未被勾选的钱包气泡不再显示

#### Scenario: 用户全选或重置筛选
- **WHEN** 用户在筛选面板中执行全选、清空或重置操作
- **THEN** 钱包勾选状态与金额筛选条件同步更新
- **AND** 图表气泡展示立即按新的筛选条件刷新
